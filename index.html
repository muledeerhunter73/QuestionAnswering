Introduction

To learn more about the AI field of natural language processing, I created a website that uses these tools and techniques to find process a middle school exam question and try to find the answer on the wikipedia.

So What is This?

This is a question answering system for middle school exam questions. It specifically seeks to find answers to who, where and when questions. It is implemented on the website muledeerhunter73.pythonanywhere.com.

Why did I do this?

Natural Language Processing fascinates me in that it allows computers in a sense to understand humans. I did this project to get my toes in the water of how natural language processing works. Answers to middle school exam questions are abundant on the web, but it can be difficult to find them in all the content.
